# 🚀 Agile Project Management System using Jira

## 📌 Project Overview
This project demonstrates the implementation of an Agile Project Management System using Jira. The objective is to manage a complete capstone project using the Scrum framework by breaking tasks into smaller units and tracking them through different workflow stages.

The system simulates a real-world project environment where tasks move through structured phases such as planning, execution, review, and completion.

---

## 🎯 Objectives
- To understand Agile and Scrum methodology in practical scenarios
- To gain hands-on experience with Jira project management tool
- To break down a large project into manageable tasks
- To track project progress in a structured and visual manner
- To improve productivity through task organization and deadlines
- To simulate real-world project workflow management

---

## 🛠 Tools Used
- Jira Software (Atlassian)
- Agile Scrum Framework

---

## 🔄 Agile Workflow

The project follows a Scrum-based workflow:

**To Do → In Progress → Approved → Done**

### 📍 Stage Description:
- **To Do**: Tasks planned but not started yet  
- **In Progress**: Tasks currently being worked on  
- **Approved**: Completed tasks under review  
- **Done**: Fully completed and verified tasks  

---

## 📊 Project Features
- Task creation and assignment
- Sprint-based planning
- Real-time progress tracking
- Deadline management
- Visual Kanban/Scrum board
- Status tracking of each task
- Structured workflow management
